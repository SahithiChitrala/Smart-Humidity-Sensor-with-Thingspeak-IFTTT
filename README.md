# 🌦️ Smart Weather Monitoring System using ESP32

## 📌 Overview

This project monitors environmental conditions using an ESP32 microcontroller and DHT22 sensor. The system measures temperature and humidity, displays readings locally, stores data for analysis, and supports cloud-based monitoring through IoT platforms.

## 🎯 Objectives

* Measure temperature and humidity in real time
* Display sensor readings on an OLED display
* Store environmental data for analysis
* Upload readings to cloud platforms
* Generate alerts under predefined conditions

## 🛠️ Components

* ESP32 Development Board
* DHT22 Temperature & Humidity Sensor
* OLED Display (SSD1306)
* SD Card Module
* Jumper Wires
* Breadboard

## ⚙️ Planned Features

### Phase 1

* Read temperature and humidity from DHT22
* Display values on Serial Monitor

### Phase 2

* Display readings on OLED screen

### Phase 3

* Log sensor data for long-term monitoring

### Phase 4

* Upload readings to ThingSpeak

### Phase 5

* Send alerts using IFTTT
